Create `lua/visual-multi/session.lua` as the Tier-2 session lifecycle module. Implement all functions exactly per the patterns in RESEARCH.md. All `require()` calls to sibling modules MUST be inside function bodies (lazy require — prevents circular dependency with init.lua).

**Session table shape** (field names are locked — higher-tier code depends on these):
```lua
{
  buf           = buf,          -- immutable after creation
  win           = win,          -- window at session start (for win-local opt restore)
  _stopped      = false,        -- sentinel: util.is_session() checks this field
  extend_mode   = initial_mode, -- boolean; matches g:Vm.extend_mode semantics
  cursors       = {},           -- cursor list (populated Phase 3+)
  _saved = {
    opts    = {},               -- { virtualedit, conceallevel, guicursor }
    keymaps = {},               -- { [lhs] = maparg_dict_or_false }
  },
  _augroup_name     = 'VM_buf_' .. buf,
  _undo_seq_before  = nil,
  _undo_lines_before = nil,
  _undo_seq_after   = nil,
}
```

**Functions to implement:**

`M.start(buf, initial_mode)` — Session start, six steps in order:
1. PITFALL-11 reentrancy guard: `if sessions[buf] then return sessions[buf] end`
2. Create session table with `_new_session(buf, initial_mode or false)`
3. Register FIRST: `sessions[buf] = session` (before any autocmd-triggering operations)
4. `_save_and_set_options(session)` — save virtualedit/conceallevel/guicursor, set session values
5. `_save_and_install_keymaps(session)` — for now, only install the `v` keymap for toggle_mode (Phase 6 adds the full keymap table)
6. `_create_augroup(session)` — creates `VM_buf_{bufnr}` with `{ clear = true }`, registers BufDelete guard
7. Emit VMEnter: `vim.api.nvim_exec_autocmds('User', { pattern = 'VMEnter', data = { bufnr = session.buf, extend_mode = session.extend_mode } })`
8. Return session

`_save_and_set_options(session)`:
- `virtualedit` (global): save `vim.o.virtualedit`, set to `'onemore'`
- `conceallevel` (window-local — NEVER use vim.bo or vim.wo without win_id): save via `vim.api.nvim_win_get_option(session.win, 'conceallevel')`, set via `vim.api.nvim_win_set_option(session.win, 'conceallevel', 0)`
- `guicursor` (global): save `vim.o.guicursor`, DO NOT modify it in Phase 2 (deferred to Phase 3 per RESEARCH.md open question 1)

`_restore_options(session)`:
- Guard: `if vim.api.nvim_win_is_valid(session.win) then` before win-local restore
- Restore all three options unconditionally (guicursor save/restore infra must be present even if modification is deferred)

`_set_vm_keymap(session, mode, lhs, rhs_fn, opts)`:
- Save existing: `local prev = vim.fn.maparg(lhs, mode, false, true)`. Check `prev.lhs ~= nil and prev.lhs ~= ''` to distinguish "map exists" from "empty dict". Store `prev` or `false` in `session._saved.keymaps[lhs]`.
- Install: `vim.keymap.set(mode, lhs, rhs_fn, { buffer = session.buf, nowait = true, silent = true })` (merge with opts)
- RESEARCH.md open question 2: wrap maparg call in `vim.api.nvim_buf_call(session.buf, function() ... end)` to ensure correct buffer context

`_save_and_install_keymaps(session)`:
- For Phase 2 scope: only install the `v` key → `function() M.toggle_mode(session) end` in normal mode
- Full keymap table installation (Esc, operator keys, etc.) is Phase 6 scope — do NOT add those here

`_restore_keymaps(session)`:
- Iterate `session._saved.keymaps`: if `prev` is truthy → `vim.fn.mapset('n', false, prev)`; if false → `pcall(vim.keymap.del, 'n', lhs, { buffer = session.buf })`
- Clear: `session._saved.keymaps = {}`

`_create_augroup(session)`:
- `vim.api.nvim_create_augroup(session._augroup_name, { clear = true })` — clear=true prevents stale accumulation (PITFALL-08)
- Register BufDelete: `once = true`, callback calls `M.stop(session.buf, { silent = true })`

`M.stop(buf, opts)`:
Stop order (RESEARCH.md pitfall 6 — set nil early):
1. Guard: `if not session then return end`
2. `sessions[buf] = nil` FIRST — prevents double-stop race (PITFALL-06 variant)
3. `session._stopped = true`
4. If not `opts.silent`: `_restore_keymaps(session)`, then `_restore_options(session)`
5. `require('visual-multi.highlight').clear(session)` — clear extmarks (safe even if no cursors)
6. `pcall(vim.api.nvim_del_augroup_by_name, session._augroup_name)` — pcall: may already be deleted
7. Emit VMLeave: `vim.api.nvim_exec_autocmds('User', { pattern = 'VMLeave', data = { bufnr = buf } })`

`M.toggle_mode(session)`:
- `session.extend_mode = not session.extend_mode`
- Phase 2: no cursor shape update (deferred to Phase 3 when rendering is built)

`M.set_mode(session, extend)`:
- `session.extend_mode = extend`

`M.set_cursor_mode(session)`:
- `if session.extend_mode then M.toggle_mode(session) end`

`M.set_extend_mode(session)`:
- `if not session.extend_mode then M.toggle_mode(session) end`

**Anti-patterns to avoid (per RESEARCH.md):**
- NEVER `vim.bo[buf].conceallevel` — silent wrong-scope write
- NEVER `vim.o.conceallevel` — does not exist
- NEVER module-level mutable session state
- NEVER `nvim_del_augroup_by_name` without pcall
- NEVER fire VMEnter before full initialization

**Required test cases** (implement ALL of these):

Category A — Session start and registry:
- `T['start returns a session table']` — call `session_mod.start(buf, false)`, assert result is table with `.buf == buf`
- `T['start registers session in init._sessions']` — after start, `require('visual-multi')._sessions[buf]` equals the returned session
- `T['start sets extend_mode from initial_mode arg']` — start with `true`, check `session.extend_mode == true`; start fresh buffer with `false`, check `false`

Category B — Session stop and teardown:
- `T['stop removes session from registry']` — start then stop; `require('visual-multi')._sessions[buf]` is nil
- `T['stop sets _stopped = true on session']` — capture session ref, then stop; check `session._stopped == true`

Category C — Option save/restore:
- `T['stop restores virtualedit to pre-session value']` — record `vim.o.virtualedit` before start; start (sets 'onemore'); stop; assert restored to original value
- `T['stop restores conceallevel to pre-session value']` — set `vim.api.nvim_win_set_option(win, 'conceallevel', 2)` before start; start; stop; assert value is 2 again

Category D — Reentrancy guard (PITFALL-11):
- `T['start on already-active session returns existing session, not a new one']` — start once, capture ref; start again on same buf; assert both calls return the same table (identity equality `==`)
- `T['start on already-active session does not double-register']` — after double-start, stop once; `_sessions[buf]` is nil (session fully removed in one stop)

Category E — Mode toggle (FEAT-03):
- `T['toggle_mode flips extend_mode from false to true']` — start with false; `session_mod.toggle_mode(session)`; assert `session.extend_mode == true`
- `T['toggle_mode flips extend_mode from true to false']` — start with true; toggle; assert false
- `T['set_cursor_mode sets extend_mode to false']` — start with true; `session_mod.set_cursor_mode(session)`; assert false
- `T['set_extend_mode sets extend_mode to true']` — start with false; `session_mod.set_extend_mode(session)`; assert true

Category F — Lifecycle events:
- `T['VMEnter fires on start']` — register autocmd on User VMEnter to capture event.data; call start; assert `captured_data.bufnr == buf`
- `T['VMLeave fires on stop']` — similar pattern for VMLeave; register autocmd, start then stop, assert captured

Category G — Keymap save/restore (PITFALL-09):
- `T['stop removes v keymap installed by session']` — after stop, `vim.fn.maparg('v', 'n', false, true)` in buffer context should have empty/nil lhs (no VM mapping)
- `T['stop restores pre-existing v keymap if one existed']` — install a user `v` mapping on the buffer; start session (VM overwrites it); stop session; assert user's original mapping is restored via maparg
