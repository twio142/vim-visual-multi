```lua
function M.is_session(arg)
  return type(arg) == 'table' and arg._stopped ~= nil
end
```
Document with a comment: `-- Sentinel: session tables always carry `_stopped` field (bool). Raw bufnr is an integer.`

**Length helpers** — Named functions so callers never use bare `#str` in position math (PITFALL-14):
```lua
function M.byte_len(s)      return #s end
function M.char_len(s)      return vim.fn.strcharlen(s) end
function M.display_width(s) return vim.fn.strdisplaywidth(s) end
```

**`M.pos2byte(buf, line, col)`** — Converts 1-indexed `(line, col)` to a 0-indexed byte offset. Uses `nvim_buf_get_offset` per LUA-03. Do NOT use `vim.fn.line2byte`:
```lua
function M.pos2byte(buf, line, col)
  -- nvim_buf_get_offset: buf, row (0-indexed) → byte offset of line start
  return vim.api.nvim_buf_get_offset(buf, line - 1) + col - 1
end
```

**`M.char_at(buf, lnum, col)`** — Returns the character at 1-indexed `(lnum, col)` in buf. Byte-safe via `vim.fn.matchstr`:
```lua
function M.char_at(buf, lnum, col)
  local line = vim.api.nvim_buf_get_lines(buf, lnum - 1, lnum, true)[1]
  if not line then return '' end
  return vim.fn.matchstr(line, string.format('\\%%%dc.', col))
end
```
